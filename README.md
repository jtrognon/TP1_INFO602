# TP1_INFO602
Machine de Turing 

## 1 - Lancement du site web

1. Extraire l'archive.
2. Ouvrir *index.html* dans un navigateur.

## 2 - Fonctionnalités présentes

- Chargement d'une machine de Turing à partir d'un fichier.
- Utilisation de plusieurs rubans (voir */mtExamples/multi_tapes.mt*).
- Visualisation de la table de transitions.
- Changement de langue. 

## 3 - Test de différentes Machines de Turing

Plusieurs fichiers sont à disposition dans */mtExamples/*.</br>
Certains fichiers sont incorrects et servent à vérifier le fontionnement de la gestion des erreurs.
